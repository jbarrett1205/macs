Create a New Resource

Resources are used to allow access to certain Makerspace assets. Every member has the main door resource added to their account upon creation, but it is expected that all other resources will require some sort of training before access is granted.

The "Secret" field is the resource key that must be passed as the "X-resource-key" header when HTTP validation requests are sent to the server. If left blank, no validation is required.

The "Cost per hour" field is for future expansion to allow the system to track resource usage such that members can be informed how much in consumables (3D printer material, welding wire/gases, etc) they have used so we can solicit payments/donations. This field is in dollars.
