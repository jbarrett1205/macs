Upload Keycards

Keycards can be uploaded in CSV (comma-separated values) format. The format is "keycard_id,comment" - any CSV fields other than the first 2 are ignored. The "keycard_id" is a hexadecimal string and the "comment" is optional and can be blank (but the column must exist in the file)

The parser skips the first line of the file as it is assumed to be headers.
